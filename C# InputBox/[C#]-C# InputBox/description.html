Introduction

A native C # InputBox as an option to use VB's Interaction.InputBox, with the possibility of using the native icons in Windows.

Um InputBox nativo C# como opção ao uso do Interaction.InputBox do VB, com a possibilidade de usar os ícones nativos no Windows.

Building the Sample

The sample project already has the class and does not have any special requirements.

O projeto de exemplo já possui a classe e não tem nenhum requerimento especial.

Description

A simple demonstration of how to implement an InputBox with C # using Windows icons. 

Uma demonstração simples de como implementar um InputBox com C# utilizando ícones Windows.

            {
                prompt.Close();
            };
            prompt.Controls.Add(painel);
            prompt.Controls.Add(confirmaracao);
            prompt.Controls.Add(cancelaracao);
            painel.Controls.Add(textLabel);
            painel.Controls.Add(icone);
            painel.Controls.Add(inputBox);
            prompt.ShowDialog();
            return inputBox.Text;
        }
    }
