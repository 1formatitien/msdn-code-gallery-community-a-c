SearchUnit - Basic Examples

These examples are extremely cut down, with very little additional styling or scripts, this is to demonstrate the most basic requirements to use search, and keep the code concise. For a more visually appealing demo, please try the 'Food and Recipes' project.

Although the examples live on HTML pages, they can of course be applied to any framework, such as MVC, Web Forms etc.

  • Single Page Searching - search box and results on one page.
  • Multiple Page Search - search box on one page and results on a different page.
  • Templates - templates used to customize appearance.
  • Filters - search filters on custom data.
  • Sorting - sorting on custom data.
  • Search Cloud - search/keyword cloud.
  • Categorization - categorizing results by location and content.
  • Spelling Correction - 'did you mean?' style spelling correction.
