Bing Maps SDK for Windows Store apps Samples demonstrates the power of Windows 8 and Bing Maps to provide an enhanced mapping experience for modern-style apps. These samples show how to create Windows Store apps that showcase the Bing Maps SDK features for both C# and JavaScript.


You must install the Bing Maps SDK for Windows Store apps and get a Bing Maps Key for Windows Store apps to run these samples.  This SDK combines the power of Windows 8 and Bing™ Maps to provide an enhanced mapping experience for modern-style apps. Developers can use this Bing Maps control to incorporate the latest road maps, aerial views, and low-angle high-resolution images into a modern-style app.

JavaScript Developers: See Developing a Windows Store app to learn how to use the Bing Maps AJAX Control 7.0 to add Bing Maps to your Window Store app.

Visual Basic, C# and C++ Developers: See Getting Started with Windows Store apps.

Important note for Visual Basic, C# and C++ Developers: For your Windows Store app project to work correctly, make sure you complete the steps in Add Bing Maps to a Windows Store app after you install the SDK.

For more information, please refer to the documentation on MSDN.

Questions? Need help? Post your questions on the Bing Maps forum for help from the product team and other community members.

System Requirements

  • Bing Maps SDK for Windows Store apps
  • Windows 8 RTM
  • Visual Studio 2012 RTM

Sample index

Bing Maps C# Pushpin Sample
The Pushpin Sample shows how to place pushpins (interactive icons) on a map using C#, XAML, and the Bing Maps SDK for Windows Store apps.
Bing Maps C# LocationTracking Sample
The Location Tracking sample shows how to acquire and display of your current position on a map by using C#, XAML, and the Bing Maps SDK for Windows Store apps.
Bing Maps C# Map Style Sample
The Map Styles sample shows how to create maps in your application with aerial imagery and road imagery by using C#, XAML, and the Bing Maps SDK for Windows Store apps.
Bing Maps C# Rotate Map Sample
The Rotate Map Sample shows how to create and rotate a bird's eye map (map with high resolution aerial imagery) by using C#, XAML, and the Bing Maps SDK for Windows Store apps.
Bing Maps C# Traffic Sample
The Traffic sample demonstrates how easy it is to superimpose a traffic layer on your map control, created using C#, XAML and the Bing Maps SDK for Metro style apps (Release Preview).
Bing Maps C# GPX Route Viewer Sample
The Bing Maps C# GPX Route Viewer Sample shows how to read in a GPX file and display its contents on the map as a route.
Bing Maps C# Map Shapes Sample
The Bing Maps C# Maps Sample shows how to add shapes to the map including polygons and polylines. It also indicates the difference between valid and invalid polygons.
Bing Maps C# Landmark Events Sample
The Bing Maps C# Landmark Events Sample shows how to handle the LandmarkTapped event.
Bing Maps C# Pixel to Location Sample
The Bing Maps C# Pixel To Location Sample shows how to convert a pixel position to a Location (latitude/longitude).
