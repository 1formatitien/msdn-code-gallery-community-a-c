Description  

This example provides a SplitView implementation of WinJS 4.4.3 library that works with Visual Studio Tools for Apache Cordova and Windows 10.
It can be useful in the first app experiment with Apache Cordova.

The model also includes:

  • Navigation of pages with animation;
  • ListView by array;
  • ToggleSwitch with save setting in localStorage;
  • Native notification alert with button;
  • Check network connection at start.
It also uses the following Apache Cordova plugins:
  • cordova-plugin-whitelist
  • cordova-plugin-network-information
  • cordova-plugin-dialogs
  • cordova-plugin-device
The app also targets Android, iOS and Windows-ARM.

Requirements

  • Windows 10;
  • Visual Studio 2017;
  • Visual Studio Tools for Apache Cordova (setup in VS 2017).

Building the Sample

  1. Download the .zip sample and extract it.
  2. In Visual Studio 2017, open the .sln file in the extracted zip.
  3. Choose a deployment target as Android or Windows-x86 and press F5.
