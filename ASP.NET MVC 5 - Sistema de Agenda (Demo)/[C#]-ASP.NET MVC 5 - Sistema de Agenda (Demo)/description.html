ASP.NET MVC 5.

Com a lançamento do Visual Studio 2013 a comunidade de desenvolvimento Web olha para a nova versão de framework Web da Microsoft com brilho nos olhos.

O ASP.NET MVC 5 veio para dominar o desenvolvimento para Web, tornando o que era bom ainda melhor.


A Demo.

Este exemplo de código em Visual C# com .NET, ASP.NET MVC 5, Twitter Bootstrap (Framework para desenvolvimento Web Front End agil), Bootswatch (Código CSS temas para o Bootstrap) e Entity Framework 5.0, Simula um pequeno sistema de agenda, onde você pode salvar seus eventos e visualizá-los

Esta palestra foi uma realização do grupo de estudantes e desenvolvedores Brasileiros totalmente presencial e gratuita.


Camadas do Sistema e suas Responsabilidades.

  • Model - Camada responsável por distribuir reflexivamente as classes e suas propriedades para as demais camadas.
  • Business - Camada responsável por aplicar regras de negócio, validações, funções básicas do sistema e tratamento de excessões.
  • Repository - Camada de conversação, ou seja responsável por transmitir os dados, operaçõs CRUD.
  • Data Access - Camada de acesso a base de dados, no caso da demo usamos o ORM EntityFramework 5.0.
